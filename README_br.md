## Instalação
1. Clone este repositório;
2. Abra o Chrome/Edge e navegue para ```Extensões```;
3. Ative o ```Modo do desenvolvedor```;
4. Clique em ```Carregar sem pacote``` e abra a pasta que contém o arquivo ```manifest.json```;
5. Continue para o [Setup](#setup)

## Setup
1. Vá para [AniList API Clients](https://anilist.co/settings/developer), crie um novo cliente e copie seu ID;
2. Renomeie o arquivo ```/src/scripts/anilist_clientid copy.js``` para ```/src/scripts/anilist_clientid.js```;
3. Abra o arquivo ```anilist_clientid.js``` e substitua "YOUR_ANILIST_CLIENT_ID" pelo seu ID;
4. Clique com o botão direito na extensão e selecione ```Opções```;
5. Clique em ```Get token```, copie o texto e cole no campo ```AniList token```;
6. Clique em ```Set token```. Você deve ver seu avatar e nome de usuário do AniList ao recarregar a página;
